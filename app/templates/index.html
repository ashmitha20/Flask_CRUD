Welcome to Flask MongoDB CRUD App

View users and modify user details in the database using CRUD operations.

Key Features

  • Create: Add new users to the database with name, email, and secure password storage.
  • Read: View all users in a clean, organized table format with essential information.
  • Update: Modify existing user information while maintaining data integrity.
  • Delete: Remove users from the system when they're no longer needed.

Technology Stack

  • Backend: Flask Python framework
  • Database: MongoDB NoSQL database
  • Authentication: Bcrypt password hashing
  • Frontend: HTML5, CSS3, Jinja2 templates
  • Deployment: Docker containerization
